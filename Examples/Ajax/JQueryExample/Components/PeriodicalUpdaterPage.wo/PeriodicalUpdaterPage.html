Synchronous Dependency Loading

The example below shows the loading of a JavaScript dependency (date selector) that is loaded by an AJAX request. The framework handles loading dependencies before initializing components returned by AJAX requests. This is pretty much the same concept as the AjaxOnDemand function in the prototype version but in practice it's a little different because it supports initialzing components through the data-wonder-id attribute.
